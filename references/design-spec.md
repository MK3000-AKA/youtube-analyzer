# YouTube 分析报告设计规范

## 设计系统

### 颜色规范

#### 背景色
| 名称 | 色值 | 用途 |
|------|------|------|
| --bg-primary | #0f0f0f | 页面背景 |
| --bg-card | #1a1a1a | 卡片背景 |
| --bg-card-hover | #242424 | 卡片悬停 |

#### 强调色
| 名称 | 色值 | 用途 |
|------|------|------|
| --accent-youtube | #ff0000 | YouTube红，主强调 |
| --accent-positive | #22c55e | 正面/积极 |
| --accent-neutral | #f59e0b | 中立/警告 |
| --accent-negative | #ef4444 | 负面/危险 |
| --accent-info | #3b82f6 | 信息/趋势 |

#### 文字色
| 名称 | 色值 | 用途 |
|------|------|------|
| --text-primary | #fff | 主标题 |
| --text-secondary | #e0e0e0 | 正文 |
| --text-muted | #888 | 辅助文字 |

### 布局规范

#### 容器
- 最大宽度: 1100px
- 水平居中: margin: 0 auto
- 内边距: 32px 24px

#### 卡片
- 圆角: 12px
- 边框: 1px solid #2a2a2a
- 内边距: 20px
- 悬停效果: border-color 变为 #ff0000

#### 间距
- 模块间距: 36px (margin-bottom)
- 卡片间距: 16px (grid gap)
- 元素间距: 12px (flex gap)

### 字体规范

| 元素 | 大小 | 粗细 | 颜色 |
|------|------|------|------|
| 页面标题 | 28px | 700 | #fff |
| 模块标题 | 18px | 700 | #fff |
| 统计数字 | 26px | 700 | #fff |
| 正文 | 14px | 400 | #e0e0e0 |
| 辅助文字 | 12px | 400 | #888 |

### 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 桌面 | > 1100px | 默认布局 |
| 平板 | 768px - 1100px | 2列网格 |
| 移动 | < 768px | 单列堆叠 |

## 9模块结构

### 模块1: 统计网格 (Stats Grid)

```
┌─────────────────────────────────────────┐
│  👁️        👍        💬        📊        📝     │
│ 92,947   4,683     364      5.0%     100    │
│ 观看次数   点赞数    总评论数   点赞率    已分析  │
└─────────────────────────────────────────┘
```

- 布局: 5列网格 (移动端2列)
- 卡片: 图标 + 数值 + 标签
- 悬停: 边框变红

### 模块2: 互动率分析 (Engagement)

```
┌─────────────────────────────────────────┐
│     ╭───────╮                           │
│    ╱    5%   ╲   互动表现                  │
│   │  ─────   │   视频获得4,683点赞...       │
│    ╲        ╱                           │
│     ╰───────╯                           │
└─────────────────────────────────────────┘
```

- 环形图: CSS conic-gradient
- 直径: 100px
- 内圆: 76px (显示百分比)

### 模块3: 视频内容摘要 (Summary)

- 纯文本块
- 背景: #1a1a1a
- 最大3行摘要

### 模块4: 评论情感分析 (Sentiment)

```
┌─────────────────────────────────────────┐
│ 正面 😊  ████████████████  45%          │
│ 中立 😐  ██████████        30%          │
│ 负面 😠  ████████          25%          │
│                                         │
│  ┌────────┬────────┬────────┐          │
│  │   45   │   30   │   25   │          │
│  │ 正面评论 │ 中立评论 │ 负面评论 │          │
│  └────────┴────────┴────────┘          │
└─────────────────────────────────────────┘
```

- 进度条: 高度10px，圆角99px
- 汇总: 3宫格卡片

### 模块5: 评论主题分布 (Topics)

```
┌──────────┬──────────┬──────────┐
│ 📦 产品反馈 │ 🔧 技术支持 │ ✨ 功能需求 │
│ 约 35%    │ 约 28%    │ 约 15%    │
│ 描述文字...│ 描述文字...│ 描述文字...│
├──────────┼──────────┼──────────┤
│ 📚 使用教程 │ 💬 社区互动 │ 📌 其他话题 │
│ 约 12%    │ 约 7%     │ 约 5%     │
│ 描述文字...│ 描述文字...│ 描述文字...│
└──────────┴──────────┴──────────┘
```

- 6宫格布局
- 图标 + 主题名 + 占比 + 描述

### 模块6: 热门评论精选 (Comments)

```
┌─────────────────────────────────────────┐
│ @用户名                    2024-03-20   │
│ 评论内容...                              │
│ 👍 123  😊 正面                          │
└─────────────────────────────────────────┘
```

- 最多5条
- 徽章类型: likes, replies, positive, technical, neutral

### 模块7: 高频关键词 (Keywords)

```
┌─────────────────────────────────────────┐
│  [keyword-1] [keyword-2] [keyword-3]    │
│  [keyword-4] [keyword-5] [...]          │
└─────────────────────────────────────────┘
```

- 词云样式
- 5级分层 (kw-1 ~ kw-5)
- 悬停放大效果

### 模块8: 核心洞察 (Insights)

```
┌──────────────┬──────────────┐
│ ✅ 积极发现    │ 📊 数据概览    │
│ 绿色左边框    │ 蓝色左边框    │
├──────────────┼──────────────┤
│ ⚠️ 警告提示    │ 🎯 内容定位    │
│ 黄色左边框    │ 红色左边框    │
└──────────────┴──────────────┘
```

- 4宫格布局
- 颜色编码: green/yellow/red/blue
- 图标 + 标题 + 描述

### 模块9: 页脚 (Footer)

- 居中文字
- 生成时间
- 数据来源
- 原视频链接
